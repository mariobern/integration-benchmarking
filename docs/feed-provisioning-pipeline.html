Reference

Pass / Fail Criteria

Session nrmse_auto_pass nrmse_conditional hit_rate_threshold
Regular (fx, equities, rates) 0.01 0.05 95%
Relaxed (commodity, metals) 0.05 0.15 85%
Extended (pre/after/overnight) 0.05 0.15 85%

Publisher Pass Logic

passes = nrmse < nrmse_auto_pass
  OR (nrmse < nrmse_conditional AND hit_rate ≥ hit_rate_threshold)

Feed Readiness Formula

feed_ready = fully_passing_count ≥ target_pub_count
# default target_pub_count = 4
# fully_passing = benchmark_passes AND uptime_passes
